Kebijakan Privasi

Kebijakan privasi ini menjelaskan bagaimana aplikasi Nomor Antrian (Android TV) mengumpulkan, menggunakan, dan melindungi data pengguna. Dengan menggunakan aplikasi ini, Anda menyetujui ketentuan dalam kebijakan privasi ini.

1. Informasi yang Dikumpulkan

Aplikasi ini dirancang untuk menampilkan nomor antrian pada layar Android TV. Aplikasi tidak mengumpulkan informasi pribadi pengguna secara langsung. Namun, aplikasi dapat mengakses data berikut:

  • Nomor antrian yang dikirim dari server/administrator.
  • Data perangkat (misalnya jenis perangkat Android TV) untuk keperluan teknis.
  • Log error atau crash report (jika terjadi) untuk perbaikan aplikasi.

2. Penggunaan Informasi

Informasi yang terkumpul hanya digunakan untuk:

  • Menampilkan nomor antrian secara real-time di Android TV.
  • Meningkatkan stabilitas dan performa aplikasi.
  • Menganalisis masalah teknis jika aplikasi mengalami error.

3. Penyimpanan dan Keamanan Data

Aplikasi ini tidak menyimpan data pribadi pengguna di perangkat maupun server. Data nomor antrian hanya ditampilkan secara sementara dan tidak disimpan setelah sesi berakhir.

4. Akses Internet

Aplikasi memerlukan koneksi internet untuk menerima dan menampilkan nomor antrian. Tidak ada data pribadi yang dikirimkan ke server selain informasi antrian.

5. Pihak Ketiga

Aplikasi ini tidak membagikan data kepada pihak ketiga untuk tujuan komersial atau iklan.

6. Perubahan Kebijakan

Kebijakan privasi ini dapat diperbarui sewaktu-waktu sesuai kebutuhan pengembangan aplikasi. Perubahan akan diumumkan melalui pembaruan aplikasi atau halaman ini.
