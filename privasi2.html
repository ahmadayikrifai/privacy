Kebijakan Privasi

Policy untuk aplikasi Password Manager

Terakhir diperbarui: (tanggal isi sendiri)

Aplikasi Password Manager menghargai privasi Anda. Dokumen ini menjelaskan bagaimana data ditangani.

1. Informasi yang Disimpan

Aplikasi ini tidak mengirimkan data pribadi Anda ke server pihak ketiga secara default. Semua data akun (nama akun, username/email, kata sandi, catatan) disimpan secara lokal di perangkat Anda dalam bentuk terenkripsi.

2. Penggunaan Informasi

Data yang disimpan digunakan hanya untuk tujuan:

  • Menyimpan dan menampilkan kredensial akun Anda.
  • Menghasilkan kata sandi kuat lewat fitur Password Generator.
  • Menyediakan impor/ekspor database terenkripsi antar perangkat bila diperlukan.

3. Enkripsi & Keamanan

Data disimpan dalam format terenkripsi (mis. AES) di perangkat. Akses data dilindungi oleh master password dan/atau biometrik jika diaktifkan. Pengembang tidak dapat memulihkan data jika Anda lupa master password karena data tidak disimpan di server pengembang.

4. Izin & Akses

Aplikasi mungkin meminta izin terbatas (mis. akses penyimpanan) hanya untuk:

  • Menyimpan file database terenkripsi (ekspor/backup).
  • Mengimpor database dari lokasi penyimpanan.

Izin tersebut tidak digunakan untuk mengumpulkan data pengguna.

5. Sinkronisasi & Pihak Ketiga

Secara standar aplikasi tidak menggunakan layanan sinkronisasi cloud. Jika suatu fitur sinkronisasi diaktifkan (mis. integrasi dengan layanan cloud), Anda akan diminta persetujuan dan diinformasikan jenis data yang akan disinkronkan.

6. Cadangan & Pemulihan

Untuk mencegah kehilangan data, disarankan melakukan ekspor/backup database terenkripsi secara berkala. Pastikan backup disimpan di tempat yang aman.

7. Batasan Tanggung Jawab

Pengembang berupaya menjaga keamanan aplikasi, namun pengguna bertanggung jawab atas:

  • Mengamankan master password.
  • Menyimpan backup pada lokasi yang aman.
  • Menggunakan versi terbaru aplikasi dan sistem operasi.

8. Perubahan Kebijakan

Kebijakan ini dapat diperbarui dari waktu ke waktu. Perubahan akan diinformasikan melalui pembaruan aplikasi atau pada halaman ini. Silakan periksa tanggal "Terakhir diperbarui".

9. Kontak

Jika ada pertanyaan mengenai Kebijakan Privasi ini, silakan hubungi:

  • Email: developer@example.com (ganti dengan email Anda)
  • Situs web: https://example.com (opsional)
